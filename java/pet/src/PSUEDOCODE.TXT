GET PETTYPE
IF PETTYPE EQUALS DOG:
	IF DOG_SPACE LESS THAN MAX:
        IF PET EXISTS:
            UPDATE INFO
        ELSE:
            CREATE PET
        SET LENGTH OF STAY
        IF LENGTH OF STAY GREATER THAN OR EQUAL TO TWO DAYS:
            IF GROOMING EQUALS YES:
                UPDATE GROOMING
        ASSIGN PET SPACE
	ELSE:
		RETURN “NO SPACE”
IF PETTYPE EQUALS CAT:
    IF CAT_SPACE LESS THAN MAX:
        IF PET EXISTS:
            UPDATE INFO
        ELSE:
            CREATE PET
        SET LENGTH OF STAY
        ASSIGN PET SPACE
